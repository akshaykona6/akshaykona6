### 🔒 Selected projects (private — happy to walk through any of these)

- **Grad Thesis — Ethernet MAC Verification**: Layered UVM testbench 
  verifying an Ethernet MAC IP core across Wishbone bus and MII PHY 
  interfaces; built 3 agents with TLM FIFO-based scoreboard validating 
  full-duplex TX/RX, CRC-32 integrity.

- **UVM Verification — RCC Module**: UVM testbench for a DTMF receiver's 
  RCC module with DPI-based C/C++ reference models; ran 18-level watermark 
  regression sweeps (1–100K) across RTL and gate-level simulations, 
  achieving 94.35% coverage.

- **SystemVerilog Verification — RCC Module**: SystemVerilog testbench 
  for a DTMF receiver's RCC module with DPI-based C/C++ reference models; 
  ran 18-level watermark regression sweeps across RTL and gate-level 
  simulations.

- **Python RTL Testbench Generation**: Python script to auto-generate 
  parameterized, synthesizable Verilog RTL and self-checking behavioral 
  testbenches for configurable multi-stage pipeline registers.

- **Perl RTL Testbench Generation**: Perl script to auto-generate 
  parameterized, synthesizable Verilog RTL and self-checking behavioral 
  testbenches for configurable multi-stage pipeline registers.
